## How to use these prompts

1. Open **Google AI Studio** → gemini-3-1-flash-image (Nano Banana Pro 2)
2. Set output: **1:1 square, 1080px**
3. Paste the prompt text
4. Download the generated image
5. Open in **Canva** or **Figma**
6. Apply the text overlay using **JetBrains Mono** font
7. Colour codes: amber `#f59e0b`, white `rgba(255,255,255,0.87)`, background `#060608`
